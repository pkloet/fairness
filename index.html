How it's calculated

Lane times are modelled using a linear mixed model: time = lane effect + race effect + noise. Each race gets its own random intercept that absorbs differences in boat class and crew speed — so only the lane signal remains. This correctly handles the fact that outer lanes appear in fewer races than inner lanes. Lanes are assigned randomly in heats (voorwedstrijden), which makes the lane effects unbiased and directly interpretable.

What the bars show

Each bar shows how many seconds slower that lane is on average compared to the fastest lane (always 0). Error bars show ±1 standard error from the model — a measure of how certain the estimate is. A narrow bar means many races contributed; a wide bar means few races and more uncertainty.

Significance symbols

The symbols next to each lane answer the question: "Is this lane significantly slower than the fastest lane?" Specifically, for each lane j the test is:

H₀: effect(lane j) − effect(fastest lane) = 0

The test statistic uses the full covariance matrix from the model, so the uncertainty of both the lane estimate and the fastest lane estimate are accounted for. The fastest lane always gets p = 1 by definition.

†  p < 0.10 — borderline, treat with caution
*  p < 0.05 — significant by the standard threshold
**  p < 0.01 — strong evidence
***  p < 0.001 — very strong evidence

The banner above the chart shows a separate overall test: whether any lane differences exist at all, using a likelihood ratio test comparing a model with lane effects to one without (χ² test, df = number of lanes − 1).

Important caveat

A significant result means the effect is detectable, not necessarily large. With many races even a small difference becomes significant. Always read the effect size in seconds alongside the p-value. Conversely, on a day with few races a real disadvantage may not reach significance — absence of evidence is not evidence of absence.
